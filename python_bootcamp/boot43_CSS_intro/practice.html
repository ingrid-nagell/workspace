CSS selector - targets a specific type of elements

  1. element selector
  2. class selector (add class keyword to html tag - multiple)
  3. id selector (ad id keyword to html tag - unique)
  4. attribute selector
  1. html{
    background: green;
    }

    h1{
    color: green;
    }

  2. .my_class{
    color: yellow;
    }

  3. #my_id{
    color: yellow;
    }

  4. target[attribute="value"]{
    color: red;
    }
